# Behaviors - gpiconsultores.com.ar

## Scroll Behaviors
- **Scroll-to-top button:** Appears when scrolled down, bottom-right corner, red circle with white up-arrow. Scrolls to top on click.
- **AOS animations:** About section columns animate in on scroll (fade-up style from AOS library).
- **Header:** Does NOT change on scroll (no sticky, no shrink). Static positioned.

## Click Behaviors
- **Nav links:** Navigate to separate pages (/servicios/, /sobre-nosotros/, /contacto/)
- **CONTACTAR button:** Links to /contacto
- **PDF buttons:** Link to PDF files (brochures in Spanish and English)
- **WhatsApp button:** Opens WhatsApp chat
- **Language flags:** Switch language via gtranslate plugin
- **Footer links:** Navigate to respective pages
- **LinkedIn icons:** Open LinkedIn profile (in new tab)

## Hover Behaviors
- **Nav links:** Likely color change on hover (standard)
- **Buttons:** Likely slight color/opacity change
- **Client logos:** No observed hover effect (carousel auto-scrolls)

## Carousel/Slider
- **Client logos:** Spectra Image Gallery in carousel mode
  - Auto-scrolling
  - Shows multiple logos at once (~7 visible)
  - Continuous scroll effect

## Responsive Behavior
- Desktop (1440px): Full layout as described
- Mobile: Hamburger menu, stacked columns, full-width sections
- Header becomes mobile menu with toggle

## Animations
- AOS (Animate On Scroll) on about section containers
- No parallax effects observed
- No scroll-snap
- No smooth scroll library (Lenis, etc.)
- Videos autoplay silently in background
